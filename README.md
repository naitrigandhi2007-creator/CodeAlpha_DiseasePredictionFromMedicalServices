# 🧠 MediPredict AI — Multi-Disease Prediction System

A machine learning web app that predicts the risk of:
- ❤️ Heart Disease  
- 🍬 Diabetes  
- 🎗️ Breast Cancer  

Built using **Streamlit + Scikit-learn + XGBoost + Plotly** and deployed on **Hugging Face Spaces**.

## 📊 Features

- Multi-disease prediction in one dashboard
- Interactive medical input UI
- Real-time risk probability
- Model comparison (SVM, LR, RF, XGBoost)
- Feature importance visualization
- Modern medical-grade UI
